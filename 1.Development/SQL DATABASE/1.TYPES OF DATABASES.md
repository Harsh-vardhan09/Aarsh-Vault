#### There are a few types of databases, all service different types of use-cases

#### NoSQL databases

1. Store data in a `schema-less` fashion. Extremely lean and fast way to store data.
2. Schema less means we can change in one row but not in other and it will not complain.
3. MongoDb is schema less mongoose lets us use schema on NodeJS level.
4. Examples - MongoDB,
