## SQL Relationship:-

- In SQL we establish relationship using foreign key.
- Primary key in one is foreign key in another to form relationship.
- Here one and many is cardinality in the relationship.
 - #### types of Relationship :-
	 1. one to one 
	 2. one to many 
	 3. many to many

---
### One to one:-
- one data is related to another data in table two.
- In one its primary key on other its foreign key.
- 1X1 relationship.

### One to Many:-
- one data in table is related to multiple data in another.
- one user can create multiple posts on Instagram.
- 1 X n relationship.

### Many to Many:-

- Many data points in one table can be interrelated to multiple data points on another table.
- Students can have many subjects and each subject can have many students.
